# Dolly Kaur — Portfolio

A personal portfolio website built with React and Vite, featuring a dark/light theme toggle.

## Sections

- **Profile** — Introduction and links to GitHub, LinkedIn, and CV
- **Projects** — Showcase of personal and freelance projects
- **Skills** — Technologies and tools
- **Contact** — Get in touch form
- **Footer** — Social links

## Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- CSS Modules
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
